# 용역계약서(일반 — 간편형) (doc-416)

> 카테고리: 용역계약 | 난이도: 하 | 예상 질문 라운드: 4회

## 용도와 법적 성격

- 소규모·단기 외주(번역, 디자인 단건, 행사 진행, 간단한 운영 대행 등)를 빠르게 계약할 때 쓰는 간편형 용역계약서다. 일반용(doc-037)의 축약판으로, 분할 지급·검수·하자보수 등 무거운 장비를 덜고 핵심 조항만 담는다.
- 법적 효력은 정식 계약서와 동일하다 — 간편하다고 효력이 약한 것이 아니라, 다루는 변수의 수가 적을 뿐이다. 업무 성격에 따라 도급(완성 목적, 민법 제664조) 또는 위임(수행 목적, 제680조)이 된다.
- 유사 문서와의 구분: 금액이 크거나(수천만 원 이상) 결과물 권리·검수가 중요한 건은 doc-037(일반용) 또는 업종 특화형(IT는 doc-198 등)으로 안내한다. 견적서·발주서만으로 진행하던 거래를 문서화하는 첫 단계로도 적합하다.

## 작성 전 확인사항

- 이 간편형으로 부족한 경우: ① 지식재산권 양도가 핵심인 창작물 외주(→ doc-037/198) ② 단계 검수·하자보수가 필요한 개발·공사(→ doc-198/201) ③ 상시 출근형 인력 제공(→ 근로계약 또는 위장도급 검토).
- 을이 개인이면 보수 지급 시 원천징수(사업소득 3.3%) 처리 주체를 미리 정한다.
- 준비물: 업무 내용 메모(또는 견적서), 기간, 대금과 입금 계좌.

## 정보 수집 단계

### 1단계 — 당사자
- **질문**: 의뢰인(갑)과 수행인(을)은 각각 어떤 형태인가요? ① 법인 ② 개인사업자 ③ 개인
- **분기**: 형태별 식별정보(법인: 법인명·대표·사업자번호 / 개인사업자: 상호·대표·사업자번호 / 개인: 성명·주소·연락처)를 받는다. 을이 개인이면 원천징수 안내를 대금 조항에 붙인다.
- **주의**: 실질이 지휘·감독 하의 종속 노동이면 근로계약 검토를 권한다(위장도급 리스크).

### 2단계 — 업무 내용과 담당자
- **질문**: 의뢰하는 업무를 알려주세요. ① 계약서에 직접 기재 ② 별도 업무명세표 첨부. 업무 담당자를 계약서에 명시할까요? ① 명시 안 함 ② "경력 [n]년 이상" 등 조건으로 명시 ③ 담당자 실명으로 명시 — 명시하는 경우 담당자 변경 시 사전 통보 의무를 넣을까요?
- **분기**: ②(별지)면 "업무의 상세는 별지 업무명세표에 따른다"로 쓰고 간인을 안내. 담당자를 실명·조건으로 명시하면 제5조에 "담당자 변경 시 [7]일 전 서면 통보 + 동등 자격자 배치" 의무가 추가된다.
- **주의**: 담당자 실명 명시는 핵심 인력 이탈 시 계약 위반 시비가 되므로, 갑 입장에서는 ②(조건 명시)+변경 통보 조합이 실용적이다.

### 3단계 — 기간과 연장
- **질문**: 계약기간의 종료를 어떻게 정하나요? ① 특정 날짜까지 ② 특정 조건(업무 완료 등) 충족 시. 기간 만료 후에는? ① 연장 없음 ② 이의 없으면 자동 연장 ③ 쌍방 동의 시 연장
- **분기**: ②(조건 종료)면 조건의 판정 기준(완료 통지 + 갑 확인)을 받는다. 자동 연장이면 "만료 [30]일 전 서면 이의 없으면 [동일 기간] 연장"으로.
- **주의**: 조건 종료형은 조건 불성취 시 계약이 무한 존속할 수 있으므로 최장 기한("늦어도 [년 월 일]까지")을 함께 두는 것이 안전하다.

### 4단계 — 대금·성공보수·추가 비용·의무 선택
- **질문**: ⑴ 대금 지급 방식은? ① 매월 정액 ② 전액 일시금 ③ 분할(계약금+잔금). ⑵ 성공보수 약정을 추가하나요? ⑶ 업무 수행 비용(교통비·재료비 등)은 누가 부담하나요? ⑷ 갑의 지급 지연 시 지연이자(패널티)를 둘까요? ⑸ 보고 의무는? (안 정함 / 요청 시 / 정기) ⑹ 추가 의무 선택: 비밀유지 / 경업·명예훼손 금지
- **분기**: ①(월 정액)이면 지급일(매월 [일]일)과 중도 종료 시 일할 정산을 넣는다. ③이면 잔금 지급 조건(업무 완료 확인 후 [일]일 내)과 완료 간주 조항(완료 통지 후 [7]일 내 이의 없으면 완료 인정)을 넣는다. 지연 패널티를 두면 연 [6~12]% 범위로(상사거래 기본 연 6%, 상법 제54조 — 최신 기준 확인 필요).
- **주의**: 간편형이라도 "완료 간주" 한 줄은 꼭 넣어라 — 갑의 확인 지연으로 잔금이 묶이는 것이 소액 외주 분쟁의 대부분이다.

## 필수 기재사항 체크리스트

- [ ] 당사자 식별정보 — 누락 시 청구·집행 상대방 특정 곤란
- [ ] 업무 내용(직접 기재 또는 별지) — 누락 시 범위 분쟁
- [ ] 계약기간(또는 종료 조건 + 최장 기한) — 누락 시 종료 시점 다툼
- [ ] 대금·지급일·계좌·완료 간주 — 누락 시 지급 교착
- [ ] 비용 부담 주체 — 누락 시 실비 정산 다툼
- [ ] 해지 조항 — 누락 시 중도 이탈 시 정산 기준 부재
- [ ] 작성일·서명란·각 1부 보관

## 작성 후 안내

- 2부 작성, 각 1부 보관. 별지 업무명세표 사용 시 간인.
- 업무 완료 통지는 이메일·문자로라도 기록을 남겨 제3조 제3항(완료 간주)의 기산점을 확보하라.
- 대금 미지급 시 이 계약서를 근거로 내용증명(용역대금 청구) → 지급명령 순으로 진행할 수 있다.
- 거래가 커지거나 결과물 권리가 중요해지면 일반용(doc-037) 또는 업종 특화형으로 갱신 체결을 권장하며, 서명 전 법률 전문가 검토를 권장한다.
