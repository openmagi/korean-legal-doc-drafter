# 재직증명서 (doc-217)

> 카테고리: 증명서 | 난이도: 하 | 예상 질문 라운드: 2회

## 용도와 법적 성격

- 근로자가 현재 해당 회사에 재직 중임을 회사가 확인해 주는 문서다. 대출 심사, 비자·관공서 제출, 이직 절차 등에서 요구된다.
- 직접 규정한 법조항은 없으나, 근로기준법 제39조(사용증명서)가 근거가 된다: 사용자는 근로자가 청구하면 재직 중에도 사용 기간·업무 종류 등을 적은 증명서를 즉시 내주어야 하고, **근로자가 요구한 사항만 적어야 한다**(불리한 평가·퇴직 사유 등을 임의로 추가하면 안 된다).
- 유사 문서와의 구분: 경력증명서(doc-304)는 퇴직자(또는 과거 경력)의 근무 사실 증명, 해촉증명서(doc-412)는 프리랜서 위촉 관계의 종료 증명이다. 재직증명서는 "지금 다니고 있음"의 증명이다.

## 작성 전 확인사항

- **이 문서로 해결되지 않는 경우**: 소득 증명이 목적이면 근로소득원천징수영수증·소득금액증명(홈택스)이 별도로 필요하다. 재직증명서에 연봉을 적는 것은 제출처가 요구할 때만 한다.
- 제출처가 지정 양식·필수 항목(예: 입사일, 직위, 발급용도)을 요구하는지 먼저 확인한다.
- 발급 주체는 회사다 — 회사 직인(또는 대표자 인)이 없으면 효력을 인정받기 어렵다.

## 정보 수집 단계

### 1단계 — 근로자 정보와 재직 사항
- **질문**: ⑴ 근로자의 성명·주민등록번호(제출처가 요구하는 경우. 생년월일로 갈음 가능)·주소를 알려주세요. ⑵ 소속 부서·직위·담당 업무와 입사일은요?
- **분기**: 제출처가 개인정보 최소화를 허용하면 주민등록번호 뒷자리는 마스킹(XXXXXX-1XXXXXX)을 권한다.
- **주의**: 재직 기간은 "입사일 ~ 현재"로 적는다. 퇴직 예정일이 확정돼 있어도 재직증명서에는 적지 않는 것이 원칙(요구 사항 외 기재 금지).

### 2단계 — 발급 용도와 회사 정보
- **질문**: ⑴ 발급 용도는? ① 금융기관 제출용 ② 관공서 제출용 ③ 회사(이직처 등) 제출용 ④ 직접 입력 ⑵ 회사의 상호·사업자등록번호·대표자·주소·연락처를 알려주세요.
- **분기**: 용도란에 선택 항목이 그대로 들어간다. 용도를 특정하면 문서 오·남용을 막을 수 있어 회사·근로자 모두에게 안전하다.
- **주의**: 금융기관 제출용은 발급일 기준 유효기간(통상 1개월 이내 발급분)을 요구하는 경우가 많다 — 제출 직전 발급을 안내한다.

## 필수 기재사항 체크리스트

- [ ] 성명·생년월일(또는 주민등록번호) — 누락 시 동일인 확인 불가
- [ ] 소속·직위·담당 업무 — 제출처 심사 항목
- [ ] 입사일(재직 기간) — 누락 시 증명서의 핵심 기능 상실
- [ ] 발급 용도 — 누락 시 오·남용 위험
- [ ] 발급일자 — 유효기간 판단 기준
- [ ] 회사 정보와 직인 — 누락 시 효력 불인정

## 작성 후 안내

- 회사 직인 날인 후 원본을 교부한다. 제출처가 여러 곳이면 용도별로 각각 발급하는 것이 안전하다.
- 사용자는 근로자의 발급 청구를 거부할 수 없고(근로기준법 제39조), 근로자가 요구하지 않은 사항(징계 이력 등)을 적어서는 안 된다.
- 허위 재직증명서 발급·사용은 사문서위조·위계에 의한 업무방해 등 형사 문제가 된다 — 재직 사실과 다르게 작성하지 않는다.
