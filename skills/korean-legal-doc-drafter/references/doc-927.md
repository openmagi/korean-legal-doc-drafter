# 근로계약서(외국인근로자) (doc-927)

> 카테고리: 근로계약 | 난이도: 상 | 예상 질문 라운드: 7회

## 용도와 법적 성격

- 외국인 근로자를 채용할 때 쓰는 근로계약서다. 외국인도 근로기준법상 근로자이며 국적을 이유로 한 차별은 금지된다(근로기준법 제6조) — 최저임금·주휴·연차·퇴직금·4대보험 등 노동관계법이 내국인과 동일하게 적용되고, 서면 명시·교부 의무(제17조)도 그대로다.
- 핵심은 **체류자격(비자)별 분기**다. E-9(비전문취업, 고용허가제)는 「외국인근로자의 고용 등에 관한 법률」(외국인고용법) 제9조에 따라 **고용노동부 표준근로계약서 양식 사용이 강제**되므로 자유 양식으로 대체할 수 없다. H-2(방문취업)는 특례고용 절차, F-2/F-4/F-5/F-6은 취업 제한이 없어 일반 근로계약서로 충분하다.
- 체류자격 범위 밖의 고용은 사용자·근로자 모두 출입국관리법 위반(제18조, 제94조 — 불법고용 형사처벌·범칙금)이 되므로, 계약서 작성 전에 반드시 체류자격·체류기간을 확인해야 한다.
- 유사 문서와의 구분: 내국인용 일반 근로계약서와 본문 구조는 같으나, 체류자격 확인 조항·언어 병기·신고 의무 안내가 추가된다.

## 작성 전 확인사항

- 이 문서로 해결되지 않는 경우:
  - **E-9 채용**: 이 문서가 아니라 고용센터를 통한 고용허가 절차 + 고용노동부 표준근로계약서(EPS 시스템 발급)를 사용해야 한다. 이 가이드는 그 경우 절차 안내용으로만 쓴다.
  - 유학생(D-2)·어학연수(D-4)는 시간제취업 허가(사전 허가, 주당 시간 한도)를 받은 범위에서만 고용 가능하다.
  - 관광(B-2)·단기방문(C-3) 등 취업 불가 체류자격은 어떤 계약서로도 적법하게 고용할 수 없다.
- 준비물: 외국인등록증(앞뒤 — 체류자격·체류기간 확인), 여권, (E-9) 고용허가서, (H-2) 특례고용가능확인서, 통장 사본. 외국인등록증의 체류기간 만료일을 계약기간 설계에 반영한다.
- 하이코리아(www.hikorea.go.kr) 또는 외국인종합안내센터(1345)에서 해당 체류자격의 취업 가능 범위를 확인할 수 있다.

## 정보 수집 단계

### 1단계 — 체류자격 확인 (최우선 분기)
- **질문**: 근로자의 체류자격(비자)은 무엇입니까? ① E-9 (비전문취업) ② H-2 (방문취업) ③ F-2/F-4/F-5/F-6 (거주·재외동포·영주·결혼이민) ④ E-7 (특정활동·전문인력) ⑤ D-2/D-4 (유학·연수) ⑥ 기타/모름
- **분기**:
  - ① E-9: **표준근로계약서 의무**(외국인고용법 제9조). 고용허가 절차(내국인 구인노력 → 고용허가서 발급 → 표준계약 체결 → 입국/사업장 배치)를 안내하고, 이 가이드의 템플릿은 보조 설명용으로만 사용.
  - ② H-2: 특례고용가능확인서를 받은 사업장만 고용 가능, 채용 후 **근로개시 신고** 필요. 계약서 자체는 일반 양식 가능.
  - ③ F 계열: 취업 제한 없음 — 내국인과 동일한 일반 근로계약서. 이 가이드 템플릿 그대로 사용.
  - ④ E-7: 직종·고용주가 사증에 특정됨. 계약서가 사증 발급·연장의 기초 서류가 되므로 직무·연봉을 사증 신청 내용과 일치시킨다.
  - ⑤ D-2/D-4: 체류자격 외 활동허가(시간제취업)를 먼저 받았는지 확인 — 허가 없으면 계약 진행 중단 안내.
  - ⑥: 외국인등록증 확인 절차를 안내하고 진행을 보류한다.
- **주의**: 체류자격 확인 없이 채용하면 불법고용으로 사용자가 3년 이하 징역 또는 벌금(출입국관리법 제94조), 향후 외국인 고용 제한 처분을 받을 수 있다.

### 2단계 — 체류기간과 계약기간
- **질문**: ① 외국인등록증상 체류기간 만료일은 언제입니까? ② 계약기간을 어떻게 정합니까? (㉠ 기간의 정함 없음 ㉡ 기간제)
- **분기**: 기간제면 종료일이 체류기간 만료일을 넘지 않게 설계하고, "체류기간이 연장되는 경우 협의하여 갱신한다" 단서를 둔다. F-5(영주)·F-6 등은 내국인과 동일하게 무기계약이 가능하다.
- **주의**: 체류기간 만료 후의 근로 제공은 그 자체로 불법체류·불법고용이 된다. 만료 60일~30일 전 연장 신청을 챙기는 안내 문구를 특약에 넣으면 실무에 도움이 된다.

### 3단계 — 언어 병기
- **질문**: 근로자가 한국어 계약서를 충분히 이해할 수 있습니까? ① 가능 ② 불충분 — 병기 언어 선택(영어/중국어/베트남어 등)
- **분기**: ②이면 계약서를 한국어-해당 언어 2단 병기로 작성하고, "해석상 차이가 있는 경우 한국어본을 기준으로 한다" 조항을 넣는다. E-9 표준계약서는 16개 언어 번역본이 고용노동부에서 제공된다.
- **주의**: 언어 병기는 법적 의무는 아니지만(E-9 제외), 근로조건을 이해하지 못한 채 서명한 계약은 "근로조건 명시" 취지가 다투어질 수 있고 분쟁 시 사용자에게 불리하다. 핵심 조건(임금·근로시간·계약기간)만이라도 병기를 강력 권장한다.

### 4단계 — 근로조건 (내국인과 동일 항목)
- **질문**: ① 근무장소·업무내용은? ② 근로시간·휴게·주휴일은? ③ 임금(구성항목 포함)·지급일·지급방법은?
- **분기**: 상시 5인 미만 사업장이면 내국인과 동일하게 연차(제60조)·가산수당(제56조)·해고제한(제23조) 미적용 — 제7조를 "법 적용 제외" 변형으로 출력한다(doc-924 참조). 최저시급(2026년 10,320원 — 최신 고시 확인 필요) 미달 설계는 차단한다.
- **주의**: "외국인이니까 최저임금 이하"는 명백한 위법이며(최저임금법 + 근로기준법 제6조), 숙식비를 임금에서 일방 공제하는 것도 위법이다 — 숙식 제공 시 비용 부담은 **별도 서면 동의**(공제 동의서)로 정하고 제6조에 명시한다.

### 5단계 — 숙식 제공
- **질문**: 기숙사·식사를 제공합니까? ① 무상 제공 ② 유상 제공(월 공제액) ③ 제공 없음
- **분기**: ②이면 항목별 금액과 "임금에서 공제하는 데 동의한다"는 명시적 동의 문구를 특약에 넣는다(근로기준법 제43조 전액불 원칙의 예외는 법령·단협 외에 근로자 동의가 필요하다는 것이 실무 처리 — 별도 공제동의서 권장). E-9는 기숙사 시설 기준(농어업 비닐하우스 숙소 금지 등) 확인이 고용허가 요건이다.
- **주의**: 과도한 숙식비 공제로 실수령액이 최저임금에 미달하는 구조는 분쟁·감독 대상이 된다.

### 6단계 — 4대보험·전용 보험
- **질문**: (자동 안내 후 확인) 보험 가입 항목을 확인합니다.
- **분기**:
  - 산재보험: 체류자격 불문 당연 적용(불법체류자도 산재보상은 인정된다는 것이 확립된 판례).
  - 건강보험: 직장가입 의무.
  - 국민연금: 상호주의(국가별 가입 면제 여부 상이 — 최신 기준 확인 필요).
  - 고용보험: 고용안정·직업능력개발사업은 당연 적용, 실업급여는 E-9/H-2의 경우 임의가입(F 계열은 당연 적용 — 최신 기준 확인 필요).
  - **E-9/H-2 전용 4대 보험**(외국인고용법): 출국만기보험(사용자 — 퇴직금 대체), 임금체불 보증보험(사용자), 귀국비용보험(근로자), 상해보험(근로자).
- **주의**: E-9 출국만기보험에 가입해도 퇴직금 산정액과 차액이 있으면 사용자가 차액을 지급해야 한다.

### 7단계 — 당사자 정보·신고 의무
- **질문**: ① 사업장명·대표자·사업자등록번호·주소·연락처 ② 근로자 성명(여권 표기)·생년월일·외국인등록번호·국적·주소·연락처
- **분기**: 채용·이직 시 신고 의무 자동 안내 — 고용허가제(E-9/H-2) 사업주는 고용 변동 사유 발생 시 15일 이내 고용센터 신고(외국인고용법 제17조), 출입국관리법상 고용 외국인의 해고·퇴직·소재불명 시 15일 이내 출입국·외국인관서 신고(제19조).
- **주의**: 성명은 여권 영문 표기와 일치시켜야 보험·세무 처리에서 불일치 오류를 막는다.

## 필수 기재사항 체크리스트

- [ ] 체류자격·외국인등록번호·체류기간 확인 기재 — 누락 시 불법고용 위험 차단 불가
- [ ] (E-9) 표준근로계약서 사용 — 자유 양식 사용 시 외국인고용법 위반
- [ ] 계약기간이 체류기간 이내인지 — 초과 시 만료 후 구간이 불법 상태
- [ ] 임금 구성·지급일·지급방법(본인 명의 계좌) — 현금 지급 관행은 체불 분쟁 시 입증 곤란
- [ ] 숙식 제공·공제에 대한 서면 동의 — 일방 공제는 전액불 원칙 위반
- [ ] 언어 병기 또는 통역 확인 문구 — 누락 시 근로조건 명시 다툼
- [ ] 5인 미만 사업장 분기(연차·가산수당 조항 변형) — 내국인과 동일 기준
- [ ] 계약서 1부 교부(제17조) — 미교부 시 500만 원 이하 벌금
